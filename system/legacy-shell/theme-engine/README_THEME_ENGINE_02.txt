LionGateOS – Theme Engine Loader · Baseline 02
=============================================

This metadata describes where the theme engine should look for:

• Icon packs
• Wallpaper packs
• Core theme engine configuration
• Default style mode

This update does not introduce runtime code yet.
It prepares a stable configuration contract that future React/Electron code will consume.

Theme Update Pack 01 – Soft Dark Default (Purple Accent)
--------------------------------------------------------

This update defines "soft-dark-default" as the primary OS theme with:
- Deep slate backgrounds
- Softer contrast
- Soft purple accent
- Improved readability over the earlier prototype

It is intended to be the first theme users see when starting LionGateOS.
