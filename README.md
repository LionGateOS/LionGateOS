# ConstructOS · OS Shell Prototype

This is a clean baseline for your ConstructOS workspace:

- OS shell with sidebar + topbar
- Estimator tool scaffold with presets and a simple draft summary
- Business tools panel hosting the BackupManager prototype
- Document generator prototype wired into the shell
- AI workspace placeholder

## Scripts

- `npm install`
- `npm run dev`

The entry file is `index.html` → `src/main.tsx` → `src/App.tsx`.
